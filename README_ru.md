> **"Единственный RLM-сервер памяти с математически доказанной безопасностью (Sentinel Lattice). Работает локально, масштабируется глобально."**

**GoMCP** — это ядро корпоративного уровня для экосистемы Syntrex AI SOC. Это невероятно быстрый, безопасный и персистентный сервер Model Context Protocol (MCP), полностью написанный на Go. GoMCP наделяет Большие Языковые Модели (LLM) постоянной эволюционирующей памятью и самомодифицирующимся контекстом, превращая стандартных текстовых агентов в самообучающийся персистентный интеллект.

## 🚀 Ключевые возможности
- 🛡️ **Примитивы Sentinel Lattice:** (TSA, CAFL, GPS...) — математически доказанные гарантии безопасности.
- ⚡ **Субмиллисекундная задержка:** Исполнение на чистом Go (с опциональными Rust-биндингами).
- 🔌 **Более 57 нативных MCP-инструментов:** Глубокая интеграция с файловой системой, shell и git из коробки.
- 💾 **Персистентная графовая память (Causal Graph):** Иерархические уровни памяти (L0-L3), работающие на базе скоростного SQLite кэша.

## ⚡ Быстрый старт

Установите GoMCP как независимый сервер памяти всего за 30 секунд:

```bash
# Установка
go install github.com/syntrex-lab/gomcp@latest

# Инициализация контекста
gomcp init

# Запуск
gomcp serve --port 9100
```

## 🧠 Сценарии использования
- **Автономные агенты:** Создавайте агентов с бесконечной, структурированной памятью.
- **Безопасный RAG:** Делайте запросы к кодовым базам с доказуемыми границами и доступом на основе ролей.
- **Локальный контекст для ИИ:** Прокачайте ваши локальные LLM (Ollama, vLLM) с помощью централизованной нервной системы контекста.
